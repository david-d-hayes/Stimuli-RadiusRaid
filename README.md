# Radius Raid - Modified for use as experimental stimuli in an online study #

This project is a modified version of Radius Raid, originally created by Jack Rugile for the 2013 js13kGames competition. This adapted version has been developed solely for non‑commercial academic research conducted by Dr. David D. Hayes, Department of Psychology, University of Limerick.

The modifications include integration with external data‑collection systems, adjustments to gameplay logic, and other changes required to support experimental research in psychology. These changes are not part of the original Radius Raid project and should not be interpreted as updates or extensions endorsed by the original author. Researchers, developers, and players are strongly encouraged to refer to, download, and/or modify the original source code (available at: https://github.com/jackrugile/radius-raid) rather than this research‑specific adaptation.

The original game is available to play at: https://jackrugile.com/radius-raid/
